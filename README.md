# 💼 Portfólio Web

Este é o meu portfólio pessoal desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de apresentar meus projetos, habilidades e evolução como desenvolvedor Front-End.

## 🚀 Sobre o projeto

Este projeto foi criado para servir como minha vitrine profissional, reunindo informações sobre mim, tecnologias que utilizo e projetos desenvolvidos.

O foco principal foi criar uma interface simples, organizada e responsiva, proporcionando uma boa experiência ao usuário.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Funcionalidades

* 📌 Apresentação pessoal
* 💻 Exibição de projetos
* 🔗 Links para GitHub e redes sociais
* 📱 Layout responsivo
